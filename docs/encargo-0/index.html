Max 8.

Este es el proyecto final con Max 8. Para lograr esta producción audiovisual se usaron como parámetros los números arrojados por un sensor de temperatura DS18B20, este sensor cambiaba la variable que contenía el color rojo y a medida de que sube la temperatura cambia la paleta de colores del shader. Luego otro parámetro creado fue gracias a la utilización de un sensor ultrasónico y que a medida que uno acercaba un objeto al sensor este reconocía una distancia muy chica y con ese valor cambiaba las formas del shader toy a unas mas triangulares, mientras que cuando uno alejaba el objeto hacía figuras circulares. Por ultimo, creé un parámetro con los valores que entregaba la música, estos valores decimales hacían que el shader vibrara y controlaba la rapidez con que hacia el loop.
